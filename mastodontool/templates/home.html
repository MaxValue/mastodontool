Welcome!

This is a website built to help users with picking a Mastodon instance.

Mastodon is a microblogging platform which currently boasts around 3.9 million users and 3.8 thousand instances. Its decentralized nature allows Mastodon instances to communicate with all other instances in the platform. The first step of joining this platform is picking an instance. This website was created with the idea in mind to aid users in pickcing the most suitable instance for them. In the Instances tab, a table of all instances and some characterisitics about them can be found. In the Statistics tab there are some interesting about the platform as a whole. Enjoy! :D
